package.json stores the project metadata, and dependencies and defines scripts. using command npm init -y
npm install packagesname will install the packages needed by bakcend
express - framework to build apis
mongoose odm(obj doc mapper) - for defining schema, so dont have to write the queries
dotenv - load environment var from .env file
cors - allows react to call backend apis, otherwise browser blocks req
jsonwebtoken - jwt auth tokens
bcryptjs - hash passwords to protect users
nodemon - development tool - restart server when files change. node server.js -> nodemon server.js

REGISTRATION FLOW

User fills signup form
        ↓
React sends POST request
        ↓
Express route receives request
        ↓
Controller validates data
        ↓
Password is hashed
        ↓
User saved in MongoDB
